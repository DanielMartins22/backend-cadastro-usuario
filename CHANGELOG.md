# Changelog

Todas as mudanças relevantes deste projeto serão documentadas neste arquivo, em português, seguindo um formato simples e claro para uso no GitHub.

## [1.0.0] - 2026-07-16

### Adicionado
- Estrutura inicial do projeto com Node.js e Express.
- API REST para gerenciamento de usuários.
- Endpoints CRUD para criação, leitura, atualização e exclusão de usuários.
- Integração com Prisma ORM para conexão com MongoDB.
- Configuração de CORS e suporte a JSON nas requisições.
- Script de desenvolvimento com modo de observação do Node.js.
- Arquivo de documentação inicial para apresentação do projeto.

### Alterado
- Organização inicial do servidor em um ponto de entrada principal.
- Definição inicial do modelo de usuário no schema do Prisma.

### Configuração
- Suporte a variáveis de ambiente através do arquivo .env.
- Estrutura inicial preparada para uso em repositório Git.

### Observações
- Este projeto foi criado com foco em aprendizagem prática de Node.js, Express, Prisma e APIs REST.
- A implementação pode servir como base para estudos, exercícios e evolução futura.
